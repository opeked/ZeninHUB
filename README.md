# Zenin Hub | Bizarre Lineage

**Zenin Hub** é um hub completo e atualizado para o jogo **Bizarre Lineage** no Roblox. Ele oferece diversas automações e utilitários para facilitar sua progressão no jogo.

---

## ✨ Funcionalidades

### 🚀 Auto Farm & Progressão
- **AutoFarm** — Farm automático de mobs próximos
- **AutoFarm History Mode** — Modo agressivo de farm (mais rápido e com mais alcance)
- **AutoQuest** — Completa quests automaticamente
- **Auto Prestige** — Prestigia automaticamente quando disponível
- **Auto World Events** — Participa de eventos mundiais automaticamente

### ⚔️ Combate
- **Kill Aura** — Ataca inimigos próximos automaticamente
- **Auto Parry** — Tenta bloquear/parry automaticamente
- **God Mode** — Mantém a vida cheia (client-side)
- **No Stun** — Evita ficar stunado
- **Auto Use Skills** — Usa skills do Stand automaticamente enquanto farma

### 🏋️ Gym & Itens
- Teleporte direto para o Gym
- **Auto Open Chest** — Abre baús automaticamente
- **Auto Sell Items** — Vende itens automaticamente
- **Auto Spin Stand** — Gira stands automaticamente

### 🧍 Player & Movimentação
- Speed (até 500)
- Jump Power (até 500)
- **Fly** com velocidade ajustável
- **Infinite Jump**
- **Noclip**
- **Infinite Stamina** (tentativa)

### 📊 Stats & Economia
- **Auto Stats** — Adiciona pontos de status automaticamente
- Auto Prestige

### 👁️ Visuals
- Player ESP
- NPC ESP
- Item ESP

### 🛠️ Utilitários
- **FPS Unlock** (até 999 FPS)
- Anti AFK
- Rejoin Server
- Server Hop
- Teleportes para Raids (Jotaro, Dio, Kira, Avdol)

---

## 📥 Como Usar

1. Copie o script completo do **Zenin Hub**.
2. Abra seu executor (recomendados abaixo).
3. Cole o script e execute enquanto estiver dentro do jogo **Bizarre Lineage**.
4. O menu aparecerá automaticamente.

### Executores Recomendados
- Delta
- Solara
- Wave
- Arceus X
- KRNL
- Fluxus

## ⚠️ Aviso Importante

- Este script é **apenas para uso educacional e pessoal**.
- O uso de scripts em jogos online pode resultar em **banimento permanente** da sua conta.
- Algumas funções (como God Mode, Infinite Stamina e remotes) funcionam de forma **client-side** e podem não ser 100% eficazes após atualizações do jogo.
- Use por sua conta e risco.

## 📌 Notas

- Algumas funções dependem de **RemoteEvents** do jogo. Caso parem de funcionar após uma atualização, o hub será atualizado.
- Recomenda-se ajustar as coordenadas de teleporte (`Locations`) caso o jogo sofra mudanças de mapa.
- O hub foi desenvolvido com foco em **facilidade de uso** e **estabilidade**.
